Deps zum Laufen bringen:
npm set arch=armv7l && npm install --tagert_arch=armv7l && ./node_modules/.bin/electron-rebuild 

Kamera rotieren über screen config oder 
v4l2-ctl --set-ctrl=rotate=90 > /etc/rc.local

Autostart für App:
/etc/xdg/autostart/launchmirror.desktop
[Desktop Entry]
Name=Run magicspiegel
Exec=npm --prefix /home/pi/Desktop/magicspiegel run launch
Type=Application
